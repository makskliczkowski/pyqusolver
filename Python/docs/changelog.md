# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-10-01
### Added
- Initial public release of QES.
- Core Algebra module (Hilbert spaces, Hamiltonians, Operators).
- NQS module (RBM, CNN, Autoregressive networks).
- Solver module (Exact Diagonalization, VMC).
- General Python utilities (Lattices, Physics helpers).
- Documentation and examples.
