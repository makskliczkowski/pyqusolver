Welcome to Quantum EigenSolver (QES)
====================================

**Quantum EigenSolver (QES)** is a high-performance Python framework for simulating quantum many-body systems. It unifies Exact Diagonalization (ED) and Neural Quantum States (NQS/VMC), providing a flexible research toolkit built on JAX and NumPy.
