# Examples

The QES repository includes a set of examples demonstrating various workflows. These can be found in the `examples/` directory.

- **[Quickstart](../quickstart.md)**: Minimal examples for ED and NQS.
- **`examples/example_session.py`**: Managing backends and global state.
- **`examples/single_particle/`**: Advanced quadratic Hamiltonian usage.
- **`examples/example_symmetries.ipynb`**: Using point and space group symmetries.

To run an example:

```bash
cd examples
python example_session.py
```
